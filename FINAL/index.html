OPEN SOURCE!!!!

basically, if something is open source,

  • anyone can see the source code (how it works)
  • anyone can propose changes

anything™ that is open source is run by the people, and not by corporations.

this also (by extension) ensures that your data is not being sold to corporations by other corporations (or at least, if it is, you know about it)
