AdNauseam is an open-source browser extension that focuses on data obfuscation. It's built on uBlock Origin, so along with incredibly accurate and fast ad and tracker detection and blocking, is has the following features:

  • when it detects an ad, it will send an HTTP request to the ad's url. in effect, it's clicking on the ad in the background.

    this confuses ad agencies (i.e. obfuscates your data) but still gives them revenue and you're still viewing the ads (maybe)

  • you can still view these ads later (and still visit them) in your "ad vault", but the main idea of data obfuscation still stands. the companies have no idea what you want to see.

  • by default, if a site respects EFF's Do Not Track standard, AdNauseam will be disabled on that page. after all, data obfuscation is its sole purpose, and if a site isn't collecting data, who cares? (you can still disable this activity in settings of course)
